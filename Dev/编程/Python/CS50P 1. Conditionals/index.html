    match¶

    类似 switch

    原：

    1
2
3
4
5
6
    if name == "Harry" or name == "Hermione" or name == "Ron":      #字符串 ==
    print("Gryffindor")
elif name == "Draco":
    print("Slytherin")
else:
    print("Who?")

    match：

    1
2
3
4
5
6
7
    match name:
    case "Harry" | "Hermione" | "Ron": 
        print("Gryffindor")
    case "Draco":
        print("Slytherin")
    case _:
        print("Who?")
    • case _: 表示else
    • | 在case中表示或
